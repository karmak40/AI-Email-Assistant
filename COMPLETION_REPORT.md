ПЕРЕХОД НА РЕАЛЬНЫЙ API
───────────────────────────────────────────────────────────────────────────

1. Получить API ключ:
   → https://platform.deepseek.com
   → Зарегистрируйтесь
   → Получите API ключ в разделе API

2. Создать .env файл в корне проекта:
   
   EXPO_PUBLIC_DEEPSEEK_API_KEY=sk_...ваш_ключ...

3. Перезагрузить приложение:
   • Нажать 'r' в терминале
   • Или закрыть/открыть Expo Go заново

4. Теперь все запросы пойдут в реальный API ✅
