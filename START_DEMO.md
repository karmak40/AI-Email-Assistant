❓ Как это работает?
────────────────────────────────────────────────────────────────────────
1. Пользователь нажимает "Попробовать демо"
2. Открывается модальное окно
3. Пользователь вводит текст письма
4. Пользователь нажимает одну из кнопок:
   - Исправить ошибки
   - Сделать официально
   - Сделать дружелюбно
5. Текст обрабатывается через AI (DeepSeek)
6. Результат отображается рядом с оригиналом

❓ Нужен ли API ключ для тестирования?
────────────────────────────────────────────────────────────────────────
НЕТ! Сейчас используется mock-режим. Все работает без ключа.

Результаты fake, но это идеально для демонстрации функциональности.

Если хотите реальных результатов:
1. Получите API ключ на https://platform.deepseek.com
2. Добавьте в .env: EXPO_PUBLIC_DEEPSEEK_API_KEY=sk_...
3. Перезагрузите приложение

❓ Где я могу увидеть примеры?
────────────────────────────────────────────────────────────────────────
Откройте файл DEEPSEEK_EXAMPLES.tsx (или прочитайте DEMO_MODE_GUIDE.md)

Там 7 полных примеров работы с различными сценариями.

❓ Какова стоимость использования API?
────────────────────────────────────────────────────────────────────────
Очень дешево! Примерно $0.0001-0.0003 за один запрос.

Детали см. в DEEPSEEK_SETUP.md

❓ Почему возникают ошибки?
────────────────────────────────────────────────────────────────────────
Обычно это:
1. Нет интернета → проверьте соединение
2. Неверный API ключ → проверьте .env файл
3. Ключ имеет ограничение → проверьте баланс на DeepSeek
