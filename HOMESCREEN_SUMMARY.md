## 🔄 ПОТОК РАБОТЫ

```
1. Пользователь открывает приложение
   ↓
2. RootNavigator проверяет сессию
   ├─ Сессия найдена
   │  └─ Показать BottomTabs (главная + 3 экрана)
   └─ Сессия не найдена
      └─ Показать AuthScreen

3. На главной экране
   ├─ Если Gmail подключен
   │  └─ Показать профиль + быстрые действия
   └─ Если Gmail не подключен
      └─ Показать приглашение подключить Gmail

4. Пользователь может
   ├─ Просмотреть входящие (📧)
   ├─ Написать письмо (✍️)
   ├─ Зайти в настройки (⚙️)
   └─ Выйти из аккаунта (красная кнопка)

5. При выходе
   └─ Вернуться на AuthScreen
```
