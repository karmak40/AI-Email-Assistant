🚀 КАК ЗАПУСТИТЬ ПРОЕКТ:

1. Откройте терминал/PowerShell

2. Перейдите в папку проекта:
   cd "d:\Projects\Flutter\AI Email Assistant\AI-Email-Assistant"

3. Запустите проект:
   npm start

4. Выберите платформу:
   - Нажмите 'a' для Android эмулятора
   - Нажмите 'w' для веб-браузера
   - Нажмите 'j' для Expo Go на телефоне
   - Отсканируйте QR код если выбрали Expo Go
